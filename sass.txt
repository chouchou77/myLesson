 scss(sass)هي لغة css مطورة يمكن فيها التعامل مع المتغيرات و العبارات الشرطية و الحلقات التكرارية و تكون مختصرة قليلا...
و لكن قبل تنفيذها يجب تحويلها إلى css عن طريق أحد المجمعات (المترجمات (مثل المجمع sass الذي نستعمله من خلال النافذة الطرفية بعد تنزيله من خلالnpm :
sass compile.scss new.css
حيث compile.scss هو إسم ملف scss و new.css هو الملف الذي نريد ان نترجم فيه الملف الأول.
المتغيرات تبدأ ب:$ 
$color: red;
mixin هو مجموعة من الاكواد لديها متغير او عدة متغيرات مع بعض:
@mixin border-par($x,$y,$z){
  -webkit-border: $x $y $z;
  -moz-border: $x $y $z;
  -ms-border: $x $y $z;
  border: $x $y $z;
}
هذا مثال حيث لدينا 3 متغيرات مستعملين في محموعة من الاكواد كل ما نحتاجه هو تضمين هذه الاكواد لعنصر ما:
div {
  @include border-par(15px,solid,black)
}
هذا الكود سيضمن كل تلك الاكواد مع القيم التي اعطيناه له.
العبارات الشرطية مثل لغات البرمجة:
@if $x < 4 {
  font-size: 15px;
}
@else if $x == 4 {
  font-size: 20px;
}
@else {
  font-size: 25px;
}
اما الحلقات التكرارية فهناك for و while:
@for $x from 1 to 6 {
  .text-#{$x} {font-size: 15 * $x px}
}
$x: 1;
while $x < 10 {
  .text-#{$x} {font-size: 15 * $x px}
}
و هناك حلقات تكراية للمصفوفات:
@each $color in red,black,blue{
  .text-#{$color} {color: $color;}
}
الاستيراد يكون بان يكون الملف الذي نريد تصدير منه الاكواد لا يام تنفيذه من خلال ان يسبق اسمه ب: _
و حين نريد استيراده داخل ملف ان لا نكتب الشرطة السفلية و لا امتداده:
@import "fileRec"
من أجل ان نضع خصائص عنصر داخل عنصر آخر:
.old {
  font-size: 15px;
  color: red;
  margin: 10px;
}
.new {
  @extend .old;
  padding: 10px;
}
سيقوم الكلاس الجديد بنسخ كل خصائص الكلاس القديم بالإضافة لخصائصه الجديدة.
